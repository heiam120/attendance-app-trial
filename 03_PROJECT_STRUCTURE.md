# Monorepo Directory Architecture Layout

- `/.agents/rules/`: Hosts global configuration rules and strict behavioral frameworks for the autonomous AI agent.
- `/public/`: Client-side workspace containing vanilla assets (index.html, styles.css, app.js).
- `/netlify/functions/`: Backend workspace processing stateless cloud endpoint requests.
- `/01_README.md`: High-level system overview and platform architecture description.
- `/05_KNOWLEDGE.md`: Live knowledge aggregation logging engineering insights.
- `/06_DECISIONS.md`: Architectural Decision Records (ADRs) tracking stack evaluations.
- `/09_DATABASE_SCHEMA.md`: Production-ready PostgreSQL structure design mapping.
- `/11_TROUBLESHOOTING.md`: Incident tracking schema mapping historical bugs and resolutions.
